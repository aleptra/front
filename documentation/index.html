Setup

Set up your environment for Front development using GIT and the CLI tool.

git clone https://github.com/aleptra/front.git make app:create

CSS

Cascading Style Sheets are used to render and style the User Interface.

<link type="text/css" href="https://front.nu/dist/front.css" rel="stylesheet">

JavaScript

FAML (Front Attribute Markup Language) and FTML (Front Template Markup Language) uses JavaScript to run.

<script src="https://front.nu/dist/front.js"></script>
Front attributes
Attribute Description
lib Load required libraries.
🛈See all available libraries

Starter layout template

The starter layout template uses the Holy Grail layout as a base and is predefined with all associated HTML5 elements.

<!doctype html>
<html lang="en">
 <head>
   <title>My application</title>
   <meta http-equiv="X-UA-Compatible" content="ie=edge;ie=10;chrome=1">
   <meta http-equiv="Content-Type" content="text/html; charset=UTF-8">
   <base env="local:/;prod:/front/">
   <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">
   <link type="text/css" href="https://www.front.nu/dist/front.css" rel="stylesheet">
   <link type="text/css" href="https://www.front.nu/dist/design/front/theme/front.css" rel="stylesheet">
   <link type="text/css" href="assets/css/theme.css" rel="stylesheet">
   <script src="https://www.front.nu/dist/front.js" lib="popup;nav;json"></script>
 </head>
 <body>
   <header></header>
   <nav></nav>
   <aside></aside>
   <main include="home.html"></main>
   <aside></aside>
   <footer></footer>
 </body>
</html>
