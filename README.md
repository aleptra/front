## Installation

### CDN

#### Latest Stable (1.0.0)

```html
<script src="https://cdn.front.nu/1.0.0/front.js"></script>
```

_(Also available: `front.min.js`)_

#### Nightly (Development)

> ⚠️ **Nightly** — The latest changes. May be unstable.

```html
<script src="https://cdn.front.nu/nightly/front.js"></script>
```

_(Also available: `front.min.js`)_
