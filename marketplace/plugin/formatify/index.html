Formatify

By Josef Gabrielsson
Description:

Formatify turns unformatted text into formatted HTML with desired indentation level and colors.

Usage:
formatify="colors:blue,yellow,orange,slategray,green;true"
Demo:
<html> <head> <title>My application</title> <link type="text/css" href="assets/css/theme.css" rel="stylesheet"> </head> <body> <main> <div class="div">Content</div> <!-- This is a comment --> </main> </body> </html>
